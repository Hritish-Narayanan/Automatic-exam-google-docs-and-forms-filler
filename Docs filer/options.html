Google Forms & Docs Auto-Answer Options

How to use:

  1. Enter your z.ai API key above and click Save.
  2. Navigate to any Google Form or Google Doc.
  3. For Forms: The extension will automatically detect questions and fill in answers.
  4. For Docs: Click the "AI Assist" button in the bottom-right corner to access AI features.
  5. Review the generated content before submitting forms or finalizing documents.

Note: This extension requires a valid z.ai API key to function.
